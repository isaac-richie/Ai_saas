# Integration Plan (Push to 9/10)

## Objective
Close the gap from 7/10 to 9/10 by strengthening pro-grade workflow, production reliability, and collaboration polish.

## Priority Roadmap (Ranked)
1. Sequence Editor v2
   - Timeline ruler with total duration
   - Drag reorder with live preview
   - Shot trim + per‑shot transitions
2. Shot Editor Drawer
   - Inline edit of camera/lens/movement/negative prompt
   - Save without leaving shot list
3. Queue + Job Status
   - Global queue view
   - Retry / cancel
   - Failure reason surface
4. Provider Integration Depth
   - Real Runway / Veo / Kling config
   - Model selector + aspect ratio mapping per provider
5. Team Collaboration
   - Shared projects
   - Comments per shot
   - Activity log
6. Performance + Resilience
   - Optimistic UI
   - Retry logic
   - Background polling + error recovery
7. Polish
   - Micro‑animations
   - Motion guidelines
   - Story‑driven empty states

## Implementation Notes
- Start with 1 + 2 for fastest UX impact.
- 3 + 4 raise credibility with pro users.
- 5 + 6 prepare for teams and production scale.
